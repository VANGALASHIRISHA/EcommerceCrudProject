
# 🛒 E-Commerce Product Management System

A dynamic web-based **E-Commerce Product Management System** developed using **JavaServer Pages (JSP), JDBC, and Oracle Database**.
This application enables efficient management of product records with full **CRUD functionality (Create, Read, Update, Delete)** and demonstrates the practical implementation of **Java web technologies and database integration**.

---

# 📌 Project Overview

This project is designed to manage product information in an **E-Commerce environment**.
It provides an easy-to-use interface where administrators can **add, view, update, and delete products**, with all data securely stored in an **Oracle Database**.

The system demonstrates **end-to-end development of a database-driven web application**, including:

* Frontend user interface
* Backend business logic
* Database connectivity using JDBC

---

# 🚀 Features

* ✅ Add new products to the database
* ✅ View and display product details dynamically
* ✅ Update existing product information
* ✅ Delete product records
* ✅ Real-time database connectivity using **JDBC**
* ✅ Dynamic web pages using **JSP**

# ⚙️ How It Works

1️⃣ User enters product details through a **JSP form**
2️⃣ JSP connects to the **Oracle Database using JDBC**
3️⃣ SQL queries perform **CRUD operations**
4️⃣ Results are displayed dynamically on the **web pages**

# ▶️ How to Run the Project

### Step 1: Install Required Software

* Install **Oracle Database**
* Install **Apache Tomcat Server**
* Install **Eclipse IDE**
* Install **Java JDK**

---

### Step 2: Import the Project

Import the project into **Eclipse IDE**.

---

### Step 3: Configure Database Connection

Update database details in **dbConnection.jsp**

Example:

```
jdbc:oracle:thin:@localhost:1521:xe
username: system
password: yourpassword
```

---

### Step 4: Run the Project

1. Start **Apache Tomcat Server**
2. Deploy the project
3. Open the browser

```
http://localhost:8080/Ecommerce-Product-Management-System
```
